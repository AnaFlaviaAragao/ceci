# ADDAC

## Input:

- A
- Selector 0
- Selector 1

## Output:

| Selector0 | Selector1 | S(Output)   |
| --------- | --------- | ----------- |
| 0         | 0         | S = A       |
| 0         | 1         | S = ACC + A |
| 1         | 0         | S = ~A      |
| 1         | 1         | S = ACC - A |
